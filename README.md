# 📝 Smart To-Do List

A modern and responsive task management web application designed to help users organize their daily tasks efficiently and track their productivity.

## 📌 Project Overview

The Smart To-Do List allows users to create and manage tasks through a clean productivity dashboard. Users can add tasks, mark them as completed, delete tasks, search for specific tasks, and filter tasks according to their status.

The application also provides a progress overview showing the total number of tasks, completed tasks, pending tasks, and overall completion percentage.

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 🔍 Search tasks
- 📋 Filter tasks:
  - All Tasks
  - Pending Tasks
  - Completed Tasks
- 📊 View task statistics
- 📈 Track completion progress
- 💾 LocalStorage support
- 🔄 Tasks remain saved after refreshing the page
- 📱 Responsive design for desktop and mobile devices
- 🎨 Modern and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Browser LocalStorage

## ⚙️ How the Application Works

### 1. Add a Task

Enter a task in the input field and click the **Add Task** button.

### 2. Complete a Task

Click the circular check button beside a task to mark it as completed.

### 3. Delete a Task

Click the delete button to remove a task from the list.

### 4. Search Tasks

Use the search bar to quickly find a particular task.

### 5. Filter Tasks

Tasks can be filtered based on their current status:

- All
- Pending
- Completed

### 6. Track Progress

The dashboard automatically updates:

- Total Tasks
- Completed Tasks
- Pending Tasks
- Completion Percentage

## 💾 Data Storage

The application uses the browser's **LocalStorage** to store task data.

This means that tasks remain available even after refreshing or reopening the webpage in the same browser.

## 🚀 How to Run the Project

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
4. Start adding and managing your tasks.
